 * compton - compositor
 * dzen2 - bar
 * ewmhstatus - bar
 * mpd - music daemon
 * ncmpcpp - music client
 * irssi - irc-client
 * dunst - notification-daemon
 * hsetroot - wallpaper
 * unclutter - hide cursor when not needed
